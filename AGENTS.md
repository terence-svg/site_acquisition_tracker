# AGENTS.md

This repository contains the specification for the Site Acquisition Tracker v0.2.

All automated agents (including Codex) must follow these rules:

1. Do not introduce CRM concepts (leads, deals, contacts, accounts, pipelines based on people).
2. The only primary record type is Site.
3. The workflow must use the fixed nine stages defined in the specification.
4. Do not add AI features, automation engines, dashboards, or analytics.
5. Do not introduce public access, external logins, or shared links.
6. External sharing must be export-based only.
7. Keep the system small, simple, and internally focused.
8. Follow the specification exactly unless explicitly instructed otherwise.

If in doubt, default to simplicity and alignment with the spec, not feature expansion.
