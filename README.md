# AMInvest Site Tracker

Internal web application for tracking live property acquisition targets through a structured, site-first workflow.

This is a private, two-user system designed to support planning-led site acquisition. It is not a CRM, not a sales pipeline, and not a general task manager.

Core principles:
- Site-first, not contact-first
- Only live sites enter the system
- Fixed acquisition workflow
- Structured data over loose notes
- Controlled external sharing without system access

The system tracks sites from Review through to Acquired using a fixed nine-stage pipeline.
